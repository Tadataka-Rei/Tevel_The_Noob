# Competitive Programming Training Log

This repository documents my journey in competitive programming, starting on **05/06/2026**, with the goal of preparing for **ICPC** and **Olympiad-style programming contests**.

## Objective

The purpose of this repository is to build strong problem-solving skills, improve algorithmic thinking, and develop the discipline required for high-level competitive programming.

I will primarily work through problems from the following problem sets:

* Codeforces Group Contests: https://codeforces.com/group/isP4JMZTix/contests

## Rules

To ensure genuine learning and skill development:

* ❌ No AI assistance for solving problems.
* ❌ No code autocomplete tools (e.g., Copilot, Cursor autocomplete, etc.).
* ✅ All solutions will be written manually.
* ✅ Editorials and external resources may only be consulted after a serious attempt has been made.
* ✅ Every accepted solution should contribute to a deeper understanding of algorithms and problem-solving techniques.

## Daily Goal

My target is to solve **at least one problem every day**.

Consistency is more important than volume. Even on busy days, I aim to make progress and maintain the habit of problem solving.

## Documentation

For each problem, I will try my best to document:

* Problem link
* Key observations
* Initial ideas
* Failed approaches (if any)
* Final solution strategy
* Complexity analysis
* Lessons learned

The goal is not only to collect accepted solutions, but also to build a personal knowledge base that I can revisit in the future.
